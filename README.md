## жҰӮиҝ°

FocuS жҳҜдёҖдёӘй«ҳжҖ§иғҪзҡ„ Windows е…ЁеұҖжҗңзҙўдёҺеҝ«йҖҹеҗҜеҠЁе·Ҙе…·пјҢжҸҗдҫӣдёӨз§Қдә’иЎҘзҡ„дәӨдә’жЁЎејҸпјҡ

| жЁЎејҸ | и§ҰеҸ‘ | з”ЁйҖ” |
|------|------|------|
| **жҗңзҙўж Ҹ** | `Alt + Space` | зұ»дјј SpotlightпјҢиҫ“е…Ҙж–Үеӯ—жЁЎзіҠжҗңзҙўеә”з”Ё/ж–Үд»¶ |
| **и¶…зә§йқўжқҝпјҲиҪ®зӣҳпјү** | жҢүдҪҸ `Ctrl + Space` | зұ»дјј LoL дҝЎеҸ·иҪ®зӣҳпјҢйј ж ҮжӢ–жӢҪж–№еҗ‘еҝ«йҖҹйҖүжӢ© |

йҮҮз”Ё **Rust + Tauri v2** жһ„е»әпјҢдҪҝз”Ёзі»з»ҹеҺҹз”ҹ WebView2пјҲдёҚжү“еҢ…жөҸи§ҲеҷЁпјүпјҢеҶ·еҗҜеҠЁ < 100msпјҢеҶ…еӯҳеҚ з”Ё < 50MBгҖӮ
